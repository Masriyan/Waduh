## Authorized Use Only

> **⚠️ IMPORTANT LEGAL NOTICE**
>
> W.A.D.U.H. Scanner is designed **exclusively for authorized security testing and research purposes**. 

### You MUST:
- Have explicit written permission from the system owner before scanning any target
- Operate within the scope of your authorization
- Follow responsible disclosure practices
- Comply with all applicable local, national, and international laws

### You MUST NOT:
- Scan systems without authorization
- Exploit vulnerabilities on production systems
- Use this tool for malicious purposes
- Share or publish sensitive findings without permission

## Reporting Security Vulnerabilities

If you discover a security vulnerability in W.A.D.U.H. Scanner itself:

1. **Do NOT** open a public GitHub issue
2. Email the maintainers at: security@example.com (replace with actual email)
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested remediation

We will acknowledge receipt within 48 hours and provide a detailed response within 7 days.

## Responsible Disclosure

If you use W.A.D.U.H. to discover vulnerabilities in WordPress sites:

1. **Report to the site owner** before any public disclosure
2. **Allow reasonable time** (90 days recommended) for remediation
3. **Coordinate disclosure** with the affected party
4. **Never exploit** vulnerabilities for personal gain
